## 🦀 Rust Workspace 模块图

```
rust-workspace/
├── ast-core/          ← AST 节点定义（跨模块共享）
├── md-parser/         ← Markdown → AST 解析器
├── ast-gen/           ← AST 代码生成工具
├── ast-renderer/      ← AST → HTML 渲染器（WASM 用）
├── wasm-engine/       ← WASM 打包层（编译到浏览器）
│     └── [release: LTO + opt-level="z" + wasm-opt -Oz]
├── image-compressor/  ← 图片压缩（Tauri 端上传前压缩）
├── ms-toc-extractor/  ← TOC 目录提取
├── ms-local-draft/    ← 本地草稿管理（Tauri SQLite）
├── ms-storage/        ← 存储抽象层
└── ms-kb-exporter/    ← 知识库导出工具
```

## 🔐 认证 & 权限模型

```
┌──────────────────────────────────────────────────────┐
│                    JWT 双层门禁                        │
│                                                      │
│  第一层：AuthMiddleware（合法 JWT 即可）                │
│  ├── guest ✅  只读：卡片浏览 / 分类 / 图谱             │
│  └── admin ✅  只读 + 写操作                           │
│                                                      │
│  第二层：RequireRole("admin")                         │
│  └── admin ✅  卡片 CRUD / 分类管理 / 边管理 / 用户注册  │
│                                                      │
│  WebSocket 认证：AUTH 消息握手（非 URL Token）           │
└──────────────────────────────────────────────────────┘
```

## 📡 WebSocket 事件协议

```
Go Server ────────────────────────────────────── Web Reader

  CARD_UPDATED  { card_id, title, excerpt }  ──▶  图谱节点刷新
  EDGE_CREATED  { source_id, target_id }     ──▶  新连线绘制
  EDGE_DELETED  { source_id, target_id }     ──▶  连线移除
  LAYOUT_UPDATED [ { id, x, y }... ]         ──▶  节点位置同步
  ERROR         { message }                  ──▶  错误提示
```

## 🚀 构建命令

| 命令              | 用途                                |
| ----------------- | ----------------------------------- |
| `make dev`        | 一键启动 Go 后端 + Tauri 桌面端     |
| `make build`      | 生产构建（Rust + 前端 + Go）        |
| `make build-wasm` | WASM 引擎构建（release + wasm-opt） |
| `make check`      | 快速检查（Rust 编译 + 前端类型）    |
| `make test`       | 运行全部测试                        |
| `make install`    | 安装前端依赖                        |
| `make clean`      | 清理构建产物                        |
